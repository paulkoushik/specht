2.1 Specht objects

2.1-1 SymmGens
‣ SymmGens( n )( attribute )

Let n be a natural number. Then SymmGens(n) returns the corresponding transpositions \((i, i+1)\) for \(i \in \{1,2,\dots,n{-}1\}\).

    
    gap> gens:= SymmGens(4);
    [ (1,2), (2,3), (3,4) ]

2.1-2 SymmGroup
‣ SymmGroup( n )( attribute )

For any natural number n, we obtain the symmetric group \(S_n\) by passing n as an argument in SymmGens(n).

    
    gap> S4:= SymmGroup(4);
    Group([ (1,2), (2,3), (3,4) ])

2.1-3 SymmRep
‣ SymmRep( lambda )( attribute )

Let \(\lambda\) be a partition of a natural number n. Then SymmGens(lambda) returns the conjugacy class representative corresponding to \(\lambda\).

    
    gap> lambda:= [2,1,1];
    [ 2, 1, 1 ]
    gap> SymmRep(lambda);
    (1,2)

2.1-4 WordPartition
‣ WordPartition( lambda )( attribute )

For a partition \(\lambda\) of a natural number n, WordPartition(lambda) returns the corresponding canonical word made of letters \(1,2,\dots,l\).

    
    gap> lambda:= [2,1,1];
    [ 2, 1, 1 ]
    gap> w:= WordPartition(lambda);
    [ 1, 1, 2, 3 ]

2.1-5 PartitionWord
‣ PartitionWord( w )( attribute )

Let w be any given word of length n. The canonical form of w can be obtained by arranging its letters in such a way that they are in a non-decreasing order. Let the canonical word be denoted by \(w_\lambda\). Then there must be a partition \(\lmabda\vdash n\) such that \(w_\lambda\) can be obtained from it, i.e., in gerneral, \(\lambda\) is the shape of the word w. The shape of any given word \(w\) is obtained by PartitionWord(w).

    
    gap> w:= [1, 1, 2, 3];
    [ 1, 1, 2, 3 ]
    gap> lambda:= PartitionWord(w);
    [ 2, 1, 1 ]
